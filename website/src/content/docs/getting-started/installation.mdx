## macOS

Download the latest release from GitHub:

<a href="https://github.com/bae-fm/bae/releases/latest/download/bae.dmg" class="sl-button primary">
  Download for macOS
</a>

1. Open the downloaded `.dmg` file
2. Drag bae to your Applications folder
3. Open bae from Applications

macOS may show a warning when you download it. You can safely open bae normally from Applications.
