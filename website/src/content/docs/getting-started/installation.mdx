## macOS

Download the latest release from GitHub:

<a href="https://github.com/baebaebaebaebae/bae/releases/latest/download/Bae.dmg" class="sl-button primary">
  Download for macOS
</a>

1. Open the downloaded `.dmg` file
2. Drag Bae to your Applications folder
3. Open Bae from Applications

### First launch

On first launch, macOS may show a security warning. This is normal for apps downloaded outside the App Store.

1. Right-click (or Control-click) bae in Applications
2. Select "Open" from the menu
3. Click "Open" in the dialog

After this, bae will open normally.

## Building from source

Bae is open source. You can build it yourself:

```bash
git clone https://github.com/baebaebaebaebae/bae
cd bae/bae
cargo build --release
```

### Build dependencies

- Rust 1.92+
- Node.js 18+ (for Tailwind CSS)
- macOS: Xcode Command Line Tools
- Linux: `libcdio`, `libtorrent-rasterbar`
